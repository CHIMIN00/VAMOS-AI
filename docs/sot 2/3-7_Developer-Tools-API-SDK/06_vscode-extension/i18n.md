## D1. Input Schema

### 1.1 i18n 설정 (I18nConfig)

```typescript
interface I18nConfig {
  default_locale: string;                  // 기본 로케일 (기본값: "ko")
  supported_locales: string[];             // 지원 로케일 ["ko", "en"]
  fallback_locale: string;                 // 폴백 로케일 (기본값: "en")
  auto_detect: boolean;                    // OS 로케일 자동 감지
  code_comment_locale?: string;            // 코드 주석 언어 (선택)
  commit_message_locale?: string;          // 커밋 메시지 언어 (선택)
}
```

### 1.2 번역 키 (TranslationKey)

```typescript
interface TranslationKey {
  namespace: string;                       // 네임스페이스 (예: "ui", "error", "tutorial")
  key: string;                             // 키 (예: "completion.loading")
  params?: Record<string, string | number>; // 치환 파라미터
}

// 번역 파일 구조 (JSON)
interface TranslationFile {
  locale: string;
  version: string;
  namespaces: {
    [namespace: string]: {
      [key: string]: string;               // "completion.loading": "코드 완성 중..."
    };
  };
}
```

### 1.3 번역 요청 (TranslateRequest)

```typescript
interface TranslateRequest {
  keys: TranslationKey[];
  target_locale: string;
  context?: string;                        // 번역 컨텍스트 힌트
}
```

## D3. Algorithm

### 3.1 i18n 프레임워크 코어

```
CLASS I18nManager:
  config: I18nConfig
  bundles: Map<string, TranslationFile>    // locale → 번역 파일
  current_locale: string
  
  FUNCTION init(config: I18nConfig):
    this.config = config
    
    // 로케일 결정
    IF config.auto_detect:
      detected = detect_os_locale()
      this.current_locale = find_best_match(detected, config.supported_locales)
    ELSE:
      this.current_locale = config.default_locale
    
    // 번역 파일 로드
    FOR locale IN config.supported_locales:
      bundle = load_translation_file(locale)
      bundles.set(locale, bundle)
  
  FUNCTION t(key: TranslationKey) -> string:
    // 1. 현재 로케일에서 조회
    value = lookup(current_locale, key.namespace, key.key)
    
    // 2. 없으면 폴백 로케일
    IF value IS NULL:
      value = lookup(config.fallback_locale, key.namespace, key.key)
      log_missing(current_locale, key)
    
    // 3. 여전히 없으면 키 자체 반환
    IF value IS NULL:
      RETURN key.key
    
    // 4. 파라미터 치환
    IF key.params:
      value = interpolate(value, key.params)
      // "{{count}}개 파일을 인덱싱 중..." → "42개 파일을 인덱싱 중..."
    
    RETURN value
  
  FUNCTION switch_locale(locale: string):
    IF locale NOT IN config.supported_locales:
      THROW I18nError("unsupported_locale", locale)
    
    current_locale = locale
    reload_ui()                            // 전체 UI 갱신
    emit_event("locale_changed", locale)
```

### 3.2 번역 키 관리

```
FUNCTION manage_translation_keys():
  // 번역 키 형식: {namespace}.{component}.{action}
  // 예: "ui.completion.loading", "error.api.timeout", "tutorial.basic.step1"
  
  namespaces = {
    "ui":       UI 텍스트 (버튼, 라벨, 메뉴)
    "error":    에러 메시지 (error_messages.md 연계)
    "tutorial": 튜토리얼 콘텐츠 (tutorials.md 연계)
    "a11y":     접근성 라벨 (accessibility.md 연계)
    "onboard":  온보딩 텍스트 (onboarding_wizard.md 연계)
    "feedback": 피드백 UI (feedback_system.md 연계)
  }
  
  // 키 등록 규칙
  RULE: 모든 사용자 표시 문자열은 번역 키 사용 필수
  RULE: 하드코딩된 문자열 금지 (린트 규칙으로 강제)
  RULE: 새 키 추가 시 ko + en 동시 등록 필수

FUNCTION validate_translations():
  FOR locale IN supported_locales:
    bundle = load_bundle(locale)
    reference = load_bundle(config.default_locale)  // ko를 기준으로
    
    missing = reference.keys() - bundle.keys()
    extra = bundle.keys() - reference.keys()
    
    IF missing.length > 0:
      REPORT("missing_keys", locale, missing)
    IF extra.length > 0:
      REPORT("orphan_keys", locale, extra)
```

### 3.3 코드 주석/커밋 메시지 언어 선택

FUNCTION get_code_locale(config: I18nConfig, context: string) -> string:
  SWITCH context:
    CASE "code_comment":
      RETURN config.code_comment_locale OR config.default_locale
    CASE "commit_message":
      RETURN config.commit_message_locale OR config.default_locale
    DEFAULT:

## D4. Error Handling

| 에러 코드 | 설명 | 복구 전략 |
|-----------|------|----------|
| I18N-E001 | 번역 파일 로드 실패 | 폴백 로케일 사용 |
| I18N-E002 | 번역 키 누락 | 폴백 로케일 → 키 자체 반환 |
| I18N-E003 | 파라미터 치환 실패 | 원본 템플릿 표시 |
| I18N-E004 | 지원하지 않는 로케일 | 기본 로케일(ko) 사용 |
| I18N-E005 | 번역 파일 포맷 오류 | 이전 버전 캐시 사용 |
