### B-1. 모델 드리프트 감지 (Model Drift Detection)

| # | 항목 | 상세 |
|---|------|------|
| 1 | **데이터 드리프트 감지** | 입력 피처의 분포 변화 모니터링. KS Test, PSI(Population Stability Index). 피처 분포가 학습 시점 대비 유의하게 변하면 경고 |
| 2 | **컨셉트 드리프트 감지** | 입력→출력 관계(시장 구조) 변화. 동일 입력에서 모델 예측과 실제 결과 괴리 증가 추적 |
| 3 | **성능 드리프트 모니터링** | 롤링 Sharpe, 승률, Precision/Recall 추이. 이동평균 대비 2σ 하락 시 "성능 열화 경고" |
| 4 | **드리프트 원인 진단** | 시장 레짐 변화? 데이터 품질 문제? 모델 과적합? 외부 충격? 원인별 대응 프로토콜 |
| 5 | **자동 재학습 트리거** | 드리프트 감지 → 자동 재학습 파이프라인 시작. 단, 재학습 결과가 기존 모델 대비 개선된 경우에만 배포 |

| 9 | **모델 은퇴(Retirement)** | 6개월 연속 벤치마크 하회 → 자동 비활성화. 은퇴 사유, 대체 모델, 영향 분석 기록 |
