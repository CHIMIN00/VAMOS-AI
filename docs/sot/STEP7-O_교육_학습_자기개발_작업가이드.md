### O-001. 적응형 학습 엔진
```
[구현 상세]
- 학습자 프로필 기반 개인화:
  ├─ 현재 지식 수준 평가 (진단 테스트)
  ├─ 학습 스타일 파악 (시각/청각/실습)
  ├─ 학습 속도 적응
  ├─ 강점/약점 분석
  └─ 목표 기반 커리큘럼

- Khanmigo 스타일 소크라테스 교수법:
  ├─ 직접 답을 주지 않고 질문으로 유도
  ├─ 힌트 단계적 제공 (1단계→2단계→3단계)
  ├─ 사고 과정 유도: "왜 그렇게 생각해?"
  └─ 격려 + 건설적 피드백

[시중 AI 비교]
- Khanmigo: K-12 교육 특화, 수학/과학 우수
- ChatGPT: 범용 질문 답변 (교육 특화 아님)
- VAMOS 차별화: 성인 전문 학습 (코딩/투자/기술) + 메모리 기반 개인화

[구현성] V1: ✅ 즉시 (프롬프트 엔지니어링)
```

### O-002. 간격 반복 (Spaced Repetition) 시스템
```
[구현 상세]
- SM-2 알고리즘 기반 (Anki 호환):
  ├─ 난이도 평가: 쉬움/보통/어려움/까먹음
  ├─ 복습 간격 자동 조정: 1일→3일→7일→14일→30일
  ├─ 망각 곡선 기반 최적 복습 시점
  └─ 일일 복습 카드 수 제한 (과부하 방지)

- AI 카드 자동 생성:
  ├─ 대화에서 핵심 개념 → 플래시카드 자동 생성
  ├─ 논문/책 읽기 → Q&A 카드
  ├─ 코딩 개념 → 코드 퀴즈 카드
  └─ 투자 지식 → 시나리오 카드

- 카드 유형:
  ├─ 기본: 앞면(질문) / 뒷면(답)
  ├─ 빈칸 채우기: "React의 ___는 상태를 관리한다"
  ├─ 이미지 오클루전: 다이어그램 일부 가림
  └─ 코드: 코드 출력 맞추기

[구현성] V1: ✅ 즉시 (SM-2 + LLM 카드 생성)
[참고] Piotr Wozniak SM-2 알고리즘, Anki
```

### O-003. 학습 경로 생성
```
[구현 상세]

  Phase 1 (2주): Python 기초 (변수, 함수, 클래스)

- 각 Phase에:
  ├─ 학습 자료 링크 (무료 우선)
  ├─ 실습 프로젝트
  ├─ 체크포인트 테스트
  └─ 예상 소요 시간
```
