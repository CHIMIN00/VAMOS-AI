## 6-A. UI 기술 계약 (STEP6+STEP7 동기화)

> D8은 SOT 스키마를 소유하지 않지만, UI 기술 선택과 관련된 문서형 계약을 정의한다.

### 6-A.1 UI 프레임워크 계약 (LOCK)

| 항목 | 결정 | 근거 |
|------|------|------|
| V1 데스크톱 | **Tauri 2.0 + React** | STEP7 S7C-053, PLAN-3.0 UPDATED |
| V2+ 웹 | **PWA (React)** 병행 | 08 §4 웹 접근성 |
| 번들 크기 | ~30MB (Tauri) | Electron ~150MB 대비 80% 절감 |
| 백엔드 통신 | Rust ↔ React (IPC) | Tauri 기본 IPC |

### 6-A.2 UI 산출물 계약

| 산출물 유형 | 설명 | 연결 |
|------------|------|------|
| Canvas | 시각화 산출물 (차트/다이어그램/인터랙티브) | 08 §7 Artifact |
| Artifacts | Action 단계 산출물 렌더링 (코드/문서/데이터) | DESIGN 2.0 (01) 표준 5단계 |
| 알림 | Guardrails 차단/비용 경고/HITL 요청 UI | 08 §10.3 Alert Priority |
| 자율 수준 UI | L0~L3 자율 수준 전환 인터페이스 | 07 §3.2.1 LOCK |

### 6-A.3 UI 메시지 확장 매핑 (STEP6/STEP7 반영)

| 시나리오 | 표시 유형 | 연결 |
|----------|----------|------|
| Guardrails 3-Layer 차단 | deny 알림 + 차단 레이어 표시 | D7 GuardrailsCheckSchema |
| HITL 개입 요청 | 승인/거부/수정 선택지 모달 | D5 HITLRequestSchema |
| Circuit Breaker 차단 | 도구 일시 중단 안내 | D5 CircuitBreakerSchema |
| Semantic Cache 히트 | 캐시 응답 표시 (비용 절감 안내) | D6 SemanticCacheSchema |
| Autonomy Level 전환 | 수준 변경 확인 UI | D7 AutonomyLevelSchema |
| Cost Downshift | 모델 강등 알림 (80%/100%) | D7 DownshiftSchema |

### 6-A.4 Pipeline S0~S8 ↔ UI 상태 크로스 매핑 (D8-M11)

> **[D8-M11 반영]** 08 DESIGN §4.6 정본 참조. 02 정본 파이프라인 상태(S0~S8)와 UI 상태(08 §4.1) 간 매핑을 아래에 기록한다.

## 8. Failure / Fallback (UI 표현 계약 — REF-only)

### 8.1 원칙
- 런타임 실패 식별은 `failure_code`(정본)로 한다. **값 목록은 D2 정본 (REF-only)**.
- D8은 `failure_code → UI 표현(문구/우선순위/표시 형태)` 매핑 규칙만 정의한다.
- fallback 전략은 `fallback_id`(정본)로 연결한다. **값 목록은 D2 정본 (REF-only)**.

### 8.2 UI 매핑 근거 (Design)
- 08 §7.1~7.3의 표 (Front Mini / ORANGE CORE / Tool)
- 08 §7.4 Copy Templates
- 08 §10.3 Alert Priority (P0/P1/P2)

### 8.3 UI 표시 연결 표 (08 §7 요약 — 값은 D2 REF)

| Layer | 정본 failure_code (D2) | UI 표시 코드 (08) | Alert Priority |
|-------|------------------------|-------------------|----------------|
| FrontMini | FM_ERR_* | FM_ERR_* | P1~P2 |
| OrangeCore | OC_I5_* | OC_ERR_* | P0~P1 |
| Tool | TL_ERR_*, MC_ERR_* | TL_ERR_*, MC_ERR_* | P1~P2 |

> 실제 값 목록은 D2 정본 참조. D8은 "매핑 규칙/역할"만 정의.
