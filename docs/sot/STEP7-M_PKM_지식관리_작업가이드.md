## Part 1: 지식 캡처 (Knowledge Capture) [10항목]

### M-001. 자동 지식 추출 파이프라인
```
[구현 상세]
- 대화에서 자동 지식 추출:
  ├─ 핵심 사실/인사이트 자동 식별 (LLM 기반)
  ├─ 결정 사항 + 이유 기록
  ├─ 코드 패턴/솔루션 추출
  ├─ 투자 분석 결과 + 근거 추출
  └─ 질문-답변 쌍 저장

- 추출 파이프라인:
  대화 → LLM 분석 → Knowledge Unit 생성 →
  ├─ 분류 (카테고리/태그)
  ├─ 중요도 평가 (1-5)
  ├─ 5-Layer 메모리 레벨 결정
  └─ 지식그래프 노드/엣지 생성

- 사용자 확인: "이 내용을 기억해둘까요?" (중요 지식)
- 자동 저장: 일정 이상 중요도는 자동 저장

[구현성] V1: ✅ 즉시
```

### M-002. 웹 클리핑 + AI 요약
```
[구현 상세]
- 웹페이지 → 지식 변환:
  ├─ URL 입력 → 콘텐츠 추출 (Readability.js)
  ├─ AI 요약: 핵심 포인트 3-5개
  ├─ 메타데이터: 제목, 저자, 날짜, URL
  ├─ 하이라이트: 중요 문장 마킹
  └─ 태그: 자동 카테고리 + 키워드

  ├─ 기술 블로그: 코드+설명 분리 저장
  ├─ 논문: Abstract+기여+한계점+관련논문
  ├─ 투자 리포트: 종목명+목표가+근거+리스크
  └─ YouTube: 자막 → 구조화 요약

[구현성] V1: ✅ 즉시 (Readability.js + LLM)
```

### M-021. 시맨틱 지식 검색
```
[구현 상세]

  ├─ 키워드 검색 (BM25): 정확한 매칭

  ├─ 그래프 검색: 관계 기반 탐색

- 검색 최적화:
  ├─ 쿼리 확장: 동의어, 관련어 자동 추가
  ├─ 리랭킹: LLM 기반 관련성 재평가
  ├─ 필터: 날짜, 카테고리, 태그, 중요도
  └─ 파셋 검색: 카테고리별 결과 그룹화

### M-022. 컨텍스트 인식 지식 추천
```
[구현 상세]
- 현재 컨텍스트 → 관련 지식 자동 추천:
  ├─ 대화 주제: "React" 대화 → React 관련 저장 지식 표시
  ├─ 코딩 중: 현재 파일 → 관련 코드 패턴/솔루션
  ├─ 투자 분석: 현재 종목 → 이전 분석 히스토리
