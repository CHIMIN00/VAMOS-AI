### P-001. 감정 인식 시스템
```
[구현 상세]
- 텍스트 감정 분석:
  ├─ 기본 감정: 기쁨, 슬픔, 분노, 불안, 놀람, 혐오, 중립
  ├─ 세부 감정: 피로, 스트레스, 좌절, 열정, 호기심, 지루함
  ├─ 한국어 감정 분석: KoBERT 기반 + LLM 보조
  ├─ 강도 측정: 1-10 척도
  └─ 컨텍스트 인식: "괜찮아" = 진짜 괜찮음 vs 강한 부정

- 음성 감정 분석 (V2):
  ├─ 톤, 피치, 속도, 볼륨 분석
  ├─ SpeechBrain / Hume AI 모델
  └─ 실시간 감정 추적

- 멀티모달 감정 (V3):
  ├─ 텍스트 + 음성 + 표정(카메라) 통합
  └─ 더 정확한 감정 추정

[활용]
- 응답 톤 자동 조절: 사용자가 화났을 때 차분하게
- 투자 의사결정 경고: "지금 감정적인 상태에서 매매하시는 건 위험할 수 있어요"
- 집중도 추정: 대화 패턴으로 피로도 감지

[프라이버시]
- 감정 데이터 로컬 전용 처리
- opt-in 명시적 동의
- 감정 데이터 외부 전송 절대 없음

[구현성] V1: ✅ 텍스트 분석 즉시 | V2: ✅ 음성 3개월
[참고] Hume AI, "Emotion Recognition" survey papers
```

### P-002. 감정 적응형 응답
```
[구현 상세]
- 감정 기반 응답 조절:
  ├─ 기쁨 → 함께 기뻐하기 + 동기 부여
  ├─ 슬픔 → 공감 + 따뜻한 톤 + 필요시 전문가 안내
  ├─ 분노 → 차분한 톤 + 사실 중심 + 감정 인정
  ├─ 불안 → 안심 + 체계적 설명 + 단계별 접근
  ├─ 피로 → 간결한 답변 + 휴식 제안
  ├─ 스트레스 → 우선순위 정리 도움 + 심호흡 제안
  └─ 좌절 → 격려 + 대안 제시 + 성과 리마인드

- 톤 조절:
  ├─ 격식 수준: 상황에 맞는 존댓말/반말
  ├─ 정보 밀도: 피로 시 핵심만 / 의욕적일 때 상세히
  └─ 유머: 가벼운 상황에서만 (강제 아님)

[구현성] V1: ✅ 프롬프트 기반 즉시
```

### P-004. 스트레스 관리 지원
```
[구현 상세]
- 스트레스 감지:
  ├─ 대화 패턴: 짧은 문장, 부정적 키워드, 빠른 타이핑
  ├─ 활동 패턴: 야근, 수면 부족, 과도한 카페인
  └─ 투자 스트레스: 포트폴리오 급락 + 잦은 시세 확인

- 스트레스 해소 도구:
  ├─ 호흡 운동 가이드 (4-7-8 호흡법, 박스 브리딩)
  ├─ 그라운딩 기법 (5-4-3-2-1 감각)
  ├─ 짧은 명상 가이드 (TTS)
  ├─ 음악 추천 (Suno/Spotify 연동)
  └─ 산책/스트레칭 리마인더

- 높은 스트레스 시:
  ├─ "잠시 쉬어가시는 건 어떨까요?"
  ├─ 전문 상담 리소스 안내 (정신건강 위기 상담 전화번호)
  └─ 비중요 알림 자동 보류

[구현성] V1: ✅ 즉시
[윤리] VAMOS는 치료사가 아님을 명시. 심각한 상황 시 전문가 연결.
```

### P-005. 감정적 투자 결정 방지
```
[VAMOS 독자 혁신 — 시중 AI에 없는 기능]
- 투자 시 감정 상태 체크:
  ├─ 매매 요청 전 감정 상태 분석

  │  "현재 스트레스가 높은 상태입니다. 감정적 매매는 평균 -2.3% 손실을
  │   초래한다는 연구가 있습니다. 내일 다시 검토하시겠어요?"
  ├─ 쿨다운 타이머: 고감정 상태에서 매매 결정 시 대기 시간
  └─ 투자 일지 감정 기록: "이 매매를 할 때 어떤 감정이었나요?"

- FOMO/패닉 감지:
  ├─ "모두 사고 있으니 나도 사야 해" → FOMO 경고
  ├─ "전부 팔아야 해" → 패닉 셀링 경고
  └─ 과거 데이터: "비슷한 상황에서 기다린 사람들의 수익률은..."

[구현성] V1: ✅ 즉시 (3-Gate Evidence Gate 연동)
```

### P-007. 공감 대화 엔진
```

  ├─ ❌ 트라우마 심층 탐구하지 않음
  └─ ❌ "힘내세요" 같은 무의미한 격려 지양

### P-010. 감정 AI 윤리 프레임워크
```
[구현 상세]
- 감정 AI 원칙:
  1. 진단하지 않음: VAMOS는 의료 전문가가 아님
  2. 프라이버시: 감정 데이터 로컬 전용

  ├─ 자해/자살 관련 키워드 감지
  ├─ 즉각 위기 상담 연결 정보 제공
  │  (한국: 자살예방상담전화 1393, 정신건강위기상담전화 1577-0199)
  └─ 해당 대화는 일반 AI 대화와 분리
