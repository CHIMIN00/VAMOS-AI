**S7I-011** | CRITICAL | V1 | 기업 분석 리포트 자동 생성 — 구조화된 분석
- 내용: 특정 기업에 대한 포괄적 분석 리포트를 LLM으로 자동 생성
- 리포트 구조:
  ```markdown
  # [기업명] 투자 분석 리포트
  생성일: 2025-02-22 | 모델: Claude 3.5 Sonnet

  ## 1. 기업 개요
  - 사업 설명, 주요 제품/서비스, 시장 위치

  ## 2. 재무 분석
  - 매출/영업이익/순이익 추이 (3년)
  - 수익성 지표: ROE, ROA, 영업이익률
  - 성장성: YoY 매출 성장률, EPS 성장률
  - 안정성: 부채비율, 유동비율, 이자보상배율

  ## 3. 밸류에이션
  - PER / PBR / EV/EBITDA (동종업계 비교)
  - DCF 간이 계산 (가정 명시)
  - 현재 가격 vs 적정 가격 범위

  ## 4. 산업/경쟁 분석
  - 산업 트렌드, 시장 규모
  - 주요 경쟁사 비교
  - SWOT 분석

  ## 5. 리스크 요인
  - 산업 리스크, 기업 고유 리스크, 매크로 리스크

  ## 6. 최근 이벤트
  - 최근 3개월 주요 뉴스/공시
  - 실적 서프라이즈 여부

  ## 7. AI 분석 요약
  ```
