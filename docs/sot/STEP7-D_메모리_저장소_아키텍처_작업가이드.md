# Part 5 — VAMOS 5계층 메모리 상세 설계 (12건)

## 5-1. 5계층 아키텍처

```
┌─────────────────────────────────────────────────┐
│ L0: 즉시 메모리 (Session Buffer)                 │
│ 저장소: 인메모리 (Python dict)                    │
│ 수명: 세션 동안만 / TTL 없음                      │
│ 용량: 최대 200K 토큰 (컨텍스트 윈도우)             │
│ 용도: 현재 대화 컨텍스트                          │
└─────────────────────────────────────────────────┘
          ↓ 세션 종료 시 승격 판단
┌─────────────────────────────────────────────────┐
│ L1: 단기 메모리 (Short-Term)                     │
│ 저장소: SQLite + JSON                            │
│ 수명: 7일 TTL / 자동 만료                         │
│ 용량: 1,000 항목                                 │
│ 용도: 최근 대화 요약, 임시 사실                    │
└─────────────────────────────────────────────────┘
```
