## 분류 기준
- **A (§6.X 구체화 가능)**: §6.X 해당 섹션에 충분한 내용이 이미 존재하여 "§6 참조" → "§6.X 참조"로 변경만 하면 됨
- **B (§6에 내용 추가 필요)**: §6.X 매핑은 가능하나 해당 섹션에 이 피쳐의 구체적 구현 가이드가 없어 내용 보강 필요
- **C (§2~§5에 직접 기재)**: §6에 해당하는 범주가 없거나, 해당 Phase 테이블에 직접 기재가 더 적절
