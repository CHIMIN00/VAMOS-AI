# VAMOS 완전성 지도 스킬

> `/completeness-map` — 오류 유형별 도구 커버리지를 한 눈에 확인

## 이 스킬이 해결하는 문제

```
문제: "이 도구가 전부야?" → "더 있는 거 아니야?" → 불안
원인: 어떤 오류가 커버되고, 어떤 오류가 안 커버되는지 한 눈에 안 보임
해결: 오류 카테고리 × 도구 매트릭스 → 빈 셀 = 0이면 COMPLETE
```

### $ARGUMENTS 가 없는 경우 (기본):

## 빈 셀 분석

각 카테고리의 Layer A~D 중 빈 셀이 있는지 확인합니다.

**빈 셀 ≠ 문제**:
- Layer A(결정론적)가 비어있어도 Layer B/C가 커버하면 OK
- Layer D(사람확인)가 비어있으면 → 모든 카테고리에 사람 프로세스 필요하진 않음
- 문제인 경우: 특정 카테고리가 Layer A~C 전부 비어있을 때

**판정:**
```
모든 카테고리에 최소 1개 Layer 커버리지 → COMPLETE
빈 카테고리 존재 → INCOMPLETE + 해당 카테고리 + 추천 도구
```
