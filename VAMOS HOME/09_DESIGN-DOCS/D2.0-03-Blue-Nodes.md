# D2.0-03 Blue Nodes

## 역할
BLUE NODE(도메인 실행 모듈)의 정의·계층 구조·라이프사이클·CORE↔NODE 계약·레지스트리 기준 정본 (버전 2.0.0, Active Canon, 최종 수정 2026-06-11).

## 핵심 섹션
- §1 정의/역할/제약: BLUE NODE는 ORANGE CORE 규칙 **상속**, **독립 실행 불가** (RULE 1.3 §4.2)
- §3 생성/활성화 라이프사이클: P0 기본 활성 / P1 후보 생성→승인→활성 / P2 기본 OFF→세션별 승인→자동 OFF
- §4 표준 구성요소: 도메인 정책(상속+오버라이드), 템플릿 세트(I-13), 메모리 사용 규칙(06), 외부 도구(E-*) 게이트(07)
- §5 CORE ↔ NODE 인터페이스 계약 (입력/출력/상태/이벤트)
- STEP7-K 보강: MCP/A2A 프로토콜 심화, 멀티에이전트 오케스트레이션 (총 86건)

## LOCK 하이라이트
- P2 도메인: "승인 기반/기본 OFF/세션 종료 시 OFF/자동 생성 금지" 문서 레벨 박제 (RULE 1.3 §3.3, §2.6)
- P1: 후보 프로필 생성까지만 자동, 활성화는 승인
- event_type은 LogEvent 통일·고유성 보장, 실패는 fallback/deny 종결
