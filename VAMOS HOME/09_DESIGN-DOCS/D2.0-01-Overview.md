# D2.0-01 Overview

## 역할
Design 2.0의 **상위 개요 + 문서 간 정본 연결 허브** (버전 2.0.0, PHASE 0 완료, 최종 수정 2026-06-11). 01~08이 Active Canon이며 프롬프트 2.0은 초기 가이드라인 참조용.

## 핵심 섹션
- §2 시스템 상위 구조: ORANGE CORE ↔ BLUE NODE, 내부(I)/외부(E)/Self-evo(S) 구분, 전체 파이프라인(입력→분석→검색→라우팅→모델→Self-check→출력)
- §3 버전 V0~V3 "설계 고정 항목" vs "확장 항목"
- §4~5 전역 공통 규칙 + 전역 레지스트리 (EventType/FailureCode/Fallback/Core Data Schemas Index)
- §6 문서 간 의존성 맵 (01~08)
- [C4] 통합본 요약 + Mermaid 다이어그램 — 전체 시스템 다이어그램의 정본
