# D2.0-02 Orange Core

## 역할
ORANGE CORE(판단/제어/감사 가능성)의 역할·경계와 표준 런타임 파이프라인(5단계 + 상태 전이)을 확정 (버전 2.0.0, Active Canon, 최종 수정 2026-06-11). 내부 기능 **I-1~I-25** 설계의 정본.

## 핵심 섹션
- I-1~I-5: 구현 직전 수준 확정 (템플릿 + 이벤트/실패/폴백 포인트)
- I-6~I-21: 최소 "명세 + 인터페이스 + 이벤트/실패 포인트"
- 단일결정 원칙(Decision Kernel): 한 시점·한 컨텍스트·한 결론을 불변 객체로 고정
- ErrorHandlingStandard: `Result<T, VamosError>` 반환 통일 (예외 throw 금지), 모듈 경계에서 VamosError 변환
- Self-check 루프: soft(자동 1회) → hard(사용자 승인) 조건

## LOCK 하이라이트
- **DEC-001 FREEZE**: VAMOS 자체 경량 프레임워크 채택, LangChain/CrewAI 등 외부 에이전트 프레임워크 import 금지
- **DEC-002 LOCK**: LangGraph는 워크플로우 오케스트레이션 전용 허용 (Gate/Decision/정책 판정 우회 금지, 노드 실행·상태 전이·체크포인트 한정)
- 결정 상태 표기: FREEZE(Design 1.0 확정) / KEEP(구현 시 선택) / DEFER(Design 1.1+ 이관)
- event_type 고유, 실패는 레지스트리 등록 후 fallback 또는 deny 귀결
