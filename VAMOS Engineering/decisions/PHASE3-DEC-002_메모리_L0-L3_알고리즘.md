### 승격 (S7D-040)
- **L0→L1**: 세션 종료 시 자동 (요약 생성 후 저장)
- **L1→L2**: 재참조 3회 이상 **OR** 사용자 확인 **OR** QoD ≥ 0.7
- **L2→L3**: 크로스 프로젝트 재참조 or 사용자 명시 저장 (프로젝트 완료 후 범용 사실)
- 스코어링: `promotion_score = access_count×0.4 + recency×0.3 + confidence×0.3`
- **V1: 자동 승격은 사용자 알림 후 확인** / V2+: 자동

### 강등/삭제 (S7D-041)
- 미참조 기간 기반 강등 + 강등 전 사용자 확인 알림 ("이 메모리를 보관할까요?")
- 예외: `pinned=true` 메모리는 강등 제외. TTL 만료 시 자동 삭제 또는 archived 전환(§2.5.3)

### 검색 (S7D-042)
- 순서: L0 → L1 → L2 → L3 / 스코어링 `relevance × recency × frequency` / 레이어당 5, 총 20 후보 → reranker → top 5
